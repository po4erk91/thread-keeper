# Code of Conduct

This project adopts the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short:

- Be considerate. Disagreements about technical choices are expected
  and welcome; personal attacks are not.
- Assume good intent. Most miscommunication is just that —
  miscommunication.
- Keep discussion on-topic for the issue / PR / discussion thread.

If someone's behavior crosses the line, contact the maintainer via the
"Report a vulnerability" link on the repository's
[Security tab](https://github.com/po4erk91/thread-keeper/security)
(the form supports general conduct reports too). All reports are
handled privately.

The full Contributor Covenant 2.1 text is the authoritative document:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
